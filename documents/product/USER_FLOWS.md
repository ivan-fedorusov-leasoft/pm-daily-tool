# Product Design

------------------------------------------------------------------------

# Developer

## Daily Update

Today's Work shows only active games assigned to the current user.

Login → Today's Work → Write Daily Notes → Mark Team Lead Attention
(optional) → Save → Done

## Suggest Change

Game Page → Suggest Change → Submit → Wait for approval

------------------------------------------------------------------------

# Manager

## Async Review

Today's Work → Review notes → Review change requests → Approve / Reject

## Live Daily

Start Daily → Everyone enters Daily Mode → Review current game → Edit if
needed → Next Game → Finish Daily

------------------------------------------------------------------------

# Shared Daily

Daily Mode includes only active games.

-   Manager controls navigation.
-   All connected users see the same game.
-   Developers may edit only their own notes.
-   Manager may edit all project data.

## Daily Host

Daily Host may be a Manager or any assigned user.

Can:
- start Daily Mode
- control Previous / Next / Finish
- edit Daily Notes during Daily
- create Change Requests

Cannot:
- directly edit protected project data unless also Manager
